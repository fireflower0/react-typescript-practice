# react-typescript-practice

A simple app using ReactJS, Typescript and Electron

node version: v16.1.0
npm version: 7.11.2

Use yarn

```
npm install -g yarn
```

## `yarn install`

Install the required libraries

## `yarn build:main`

Compiling the main process

## `yarn build:render`

Render process compilation

## `yarn start`

Start Electron
